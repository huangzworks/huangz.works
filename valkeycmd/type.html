TYPE

TYPE命令可以以字符串形式返回给定键的类型（也即是它在数据库中关联了什么类型的值/数据结构）：

TYPE key

命令可能返回的值包括：

  • “string”

  • “hash”

  • “list”

  • “set”

  • “zset”

  • “stream”

它们分别对应字符串键、哈希键、列表键、集合键、有序集合键和流键；若命令返回”none”则表示给定键不存在。

以下是一些使用TYPE命令获取键类型的例子：

127.0.0.1:6379> TYPE User:huangz:profile
hash
127.0.0.1:6379> TYPE MessageQueue:511500
list
127.0.0.1:6379> TYPE not-exists-key
none

其他信息

TYPE命令从Valkey 1.0版本开始可用，其复杂度为O(1)。
