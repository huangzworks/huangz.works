UNLINK

UNLINK命令跟DEL命令一样，可以从数据库中移除给定的一个或多个键，并返回实际被移除的键数量为结果：

UNLINK key [key] [...]

UNLINK命令跟DEL命令不一样的是，DEL命令会在解除数据库与键关联的同时，同步释放键占用的内存；相反地，UNLINK命令只会解除数据库与键的关联，而释放键占用内存的具体工作则交给后台线程来完成，以此来实现异步的键移除操作。

以下代码展示了如何使用UNLINK命令移除键k1、k2和k3：

127.0.0.1:6379> SET k1 ""
OK
127.0.0.1:6379> SET k2 ""
OK
127.0.0.1:6379> UNLINK k1 k2 k3
(integer) 2

其他信息

UNLINK命令从Valkey 4.0版本开始可用，其复杂度为O(N)，其中N为被移除键的数量。
